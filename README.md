Society Dashboard

Society Dashboard is a responsive web application designed for managing and viewing society-related information. This project is built using Tailwind CSS, DaisyUI, JavaScript, and Handlebars.js, making it modern, lightweight, and easy to use.

Features

Responsive Design: Mobile-first layout that works seamlessly on desktop, tablet, and mobile devices.

Dynamic Content: Uses Handlebars.js for templating dynamic society data.

Styled with Tailwind CSS & DaisyUI: Clean, modern design with reusable components like buttons, cards, and navbar.

Interactive Navbar: Hamburger menu for mobile screens.

User-Friendly UI: Simple and intuitive interface for managing society information.

Technologies Used

Tailwind CSS – for utility-first styling.

DaisyUI – for ready-made UI components.

JavaScript – for DOM manipulation and interactivity.

Handlebars.js – for dynamic HTML templates.

Purpose

This project is part of my college assignment to demonstrate front-end web development skills, including responsive design, UI components, and dynamic content rendering using Handlebars.js.
